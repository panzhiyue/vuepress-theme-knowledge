# 1.函数列表

# 1.1. PostgreSQL PostGIS Geometry / Geography / Box Types

本节列出了PostGIS安装的PostgreSQL数据类型。注意我们描述了这些的铸造行为，这在设计自己的功能时非常重要。

Cast是指将一种类型强制转换为另一种类型。PostgreSQL在大多数数据库中都是独一无二的，它允许您定义自定义类型的转换行为和用于转换的函数。可以将强制转换指定为自动，在这种情况下，您不必执行CAST（myfoo为otherfootype）或myfoo :: otherfootype，如果您将其提供给仅适用于其他类型并且有自动强制转换的函数为了它。

依赖自动强制转换行为的危险在于你有一个重载函数，比如一个拿一个box2d，一个拿一个box3d而不是几何。会发生的是，两个函数同样适用于几何体，因为几何体具有两者的自动播放功能 - 因此最终会出现模糊的函数错误。要强制PostgreSQL选择，你做一个CAST（mygeom As box3d）或mygeom :: box3d。

至少从PostgreSQL 8.3开始 - 一切都可以CAST到文本（大概是因为神奇的未知类型），所以没有定义的CASTS需要为你提供CAST一个对象到文本。

box2d在新窗口打开 - 由x min，ymin，xmax，ymax组成的框。通常用于返回几何体的2d封闭盒。

box3d在新窗口打开 - 由x min，ymin，zmin，xmax，ymax，zmax组成的框。通常用于返回几何体或几何体集合的3d范围。

geometry在新窗口打开 - 平面空间数据类型。

geometry_dump在新窗口打开 - 具有两个字段的空间数据类型 - geom（保存几何对象）和path []（保存转储对象内几何体位置的一维数组。）

geography在新窗口打开 - Ellipsoidal空间数据类型。

＆>在新窗口打开 - TRUE如果A'边界框重叠或位于B的右边，则返回。

>>在新窗口打开 - TRUE如果A的边界框严格在B的右边，则返回。

# 1.8. 空间关系和测量

ST_3DClosestPoint在新窗口打开 - 返回g1上最接近g2的三维点。这是3D最短线的第一点。

ST_3DDistance在新窗口打开 - 对于几何类型返回投影单位中两个几何之间的三维笛卡尔最小距离（基于空间参考）。

ST_3DDWithin在新窗口打开 - 对于3d（z）几何类型如果两个几何3d距离在单位数内，则返回true。

ST_3DDFullyWithin在新窗口打开 - 如果所有3D几何都在彼此指定的距离内，则返回true。

ST_3DIntersects在新窗口打开 - 如果几何图形在3d中“空间相交”，则返回TRUE - 仅针对点和线串

ST_3DLongestLine在新窗口打开 - 返回两个几何之间的三维最长直线

ST_3DMaxDistance在新窗口打开 - 对于几何类型返回投影单位中两个几何之间的三维笛卡尔最大距离（基于空间参考）。

ST_3DShortestLine在新窗口打开 - 返回两个几何之间的三维最短线

ST_Area在新窗口打开 - 如果是多边形或多边形，则返回曲面的区域。对于“几何”类型区域是SRID单位。对于“地理”区域是平方米。

ST_Azimuth在新窗口打开 - 返回基于北方的方位角，作为从点A到点B的垂直方向顺时针测量的弧度角。

ST_Centroid在新窗口打开 - 返回几何的几何中心。

ST_ClosestPoint在新窗口打开 - 返回g1上最接近g2的二维点。这是最短线的第一点。

ST_Contains在新窗口打开 - 当且仅当B的外部没有位于A的外部时，返回true，并且B内部的至少一个点位于A的内部。

ST_ContainsProperly在新窗口打开 - 如果B与A的内部相交但不与边界（或外部）相交，则返回true。A本身不包含正确，但确实包含自身。

ST_Covers在新窗口打开 - 如果几何B中没有点在几何A之外，则返回1（TRUE）

ST_CoveredBy在新窗口打开 - 如果几何/地理A中没有点在几何/地理B之外，则返回1（TRUE）

ST_Crosses在新窗口打开 - 返回TRUE提供的几何是否有一些共同的内部点，但不是全部。

ST_LineCrossingDirection在新窗口打开 - 给定2个线串，返回-3到3之间的数字，表示什么样的交叉行为。0没有穿越。

ST_Disjoint在新窗口打开 - 如果几何不“空间相交”，则返回TRUE - 如果它们不共享任何空间。

ST_Distance在新窗口打开 - 对于几何类型返回投影单位中两个几何之间的二维笛卡尔最小距离（基于空间参考）。对于地理类型，默认返回两个地理位置之间的球形最小距离（以米为单位）。

ST_HausdorffDistance在新窗口打开 - 返回两个几何之间的Hausdorff距离。基本上衡量两种几何形状的相似或不同之处。单位是几何的空间参照系的单位。

ST_MaxDistance在新窗口打开 - 以投影单位返回两个几何之间的二维最大距离。

ST_Distance_Sphere在新窗口打开 - 返回两个lon / lat几何之间的最小距离（以米为单位）。使用球形地球，半径6370986米。比ST_Distance_Spheroid ST_Distance_Spheroid快，但不太准确。1.5之前的PostGIS版本仅针对点实施。

ST_Distance_Spheroid在新窗口打开 - 返回给定特定椭球体的两个lon / lat几何之间的最小距离。1.5之前的PostGIS版本仅支持点。

ST_DFullyWithin在新窗口打开 - 如果所有几何都在彼此的指定距离内，则返回true

ST_DWithin在新窗口打开 - 如果几何在彼此的指定距离内，则返回true。对于几何单位，在空间参考的单位中，对于地理单位以米为单位，测量默认为use_spheroid = true（在球体周围测量），为了更快检查，use_spheroid = false沿球体测量。

ST_Equals在新窗口打开 - 如果给定的几何表示相同的几何，则返回true。方向性被忽略。

ST_HasArc在新窗口打开 - 如果几何或几何集合包含圆形字符串，则返回true

ST_Intersects在新窗口打开 - 如果Geometries / Geography“在2D中空间相交” - （共享空间的任何部分）则返回TRUE，如果它们不共享则返回FALSE（它们是Disjoint）。对于地理位置 - 公差为0.00001米（因此任何关闭的点都被视为相交）

ST_Length在新窗口打开 - 如果它是线串或多线串，则返回几何的2d长度。几何以空间参考为单位，地理以米为单位（默认球体）

ST_Length2D在新窗口打开 - 如果它是线串或多线串，则返回几何的二维长度。这是别名ST_Length

ST_3DLength在新窗口打开 - 如果它是线串或多线串，则返回几何的三维或二维长度。

ST_Length_Spheroid在新窗口打开 - 计算椭圆体上线串/多线串的2D或3D长度。如果几何的坐标是经度/纬度并且需要长度而没有重新投影，这将非常有用。

ST_Length2D_Spheroid在新窗口打开 - 计算椭圆体上线串/多线串的2D长度。如果几何的坐标是经度/纬度并且需要长度而没有重新投影，这将非常有用。

ST_3DLength_Spheroid在新窗口打开 - 计算椭圆体上几何体的长度，并考虑高程。这只是ST_Length_Spheroid的别名。

ST_LongestLine在新窗口打开 - 返回两个几何的2维最长线在新窗口打开点。该函数仅返回函数找到的第一个最长行（如果多于一行）。返回的行将始终以g1开头并以g2结尾。此函数返回的行的长度始终与g1和g2的st_maxdistance返回的长度相同。

ST_OrderingEquals在新窗口打开 - 如果给定的几何表示相同的几何，并且点的方向顺序相同，则返回true。

ST_Overlaps在新窗口打开 - 如果几何体共享空间，则返回TRUE，具有相同的维度，但彼此不完全包含。

ST_Perimeter在新窗口打开 - 返回ST_Surface或ST_MultiSurface几何或地理边界的长度测量值。（多边形，多边形）。几何测量以空间参考为单位，地理以米为单位。

ST_Perimeter2D在新窗口打开 - 如果是多边形或多边形，则返回几何的二维周长。这是ST_Perimeter的别名。

ST_3DPerimeter在新窗口打开 - 如果是多边形或多边形，则返回几何的三维边界。

ST_PointOnSurface在新窗口打开 - 返回POINT保证躺在表面上。

ST_Project在新窗口打开 - POINT使用以米为单位的距离和以弧度表示的方位角（方位角）返回从起点开始投影的值。

ST_Relate在新窗口打开 - 如果此Geometry在空间上与anotherGeometry相关，则返回true，方法是测试由intersectionMatrixPattern中的值指定的两个几何的内部，边界和外部之间的交点。如果未传入intersectionMatrixPattern，则返回与2个几何相关的最大intersectionMatrixPattern。

ST_RelateMatch在新窗口打开 - 如果intersectionMattrixPattern1表示intersectionMatrixPattern2，则返回true

ST_ShortestLine在新窗口打开 - 返回两个几何之间的二维最短线

ST_Touches在新窗口打开 - 返回TRUE几何是否至少有一个共同点，但它们的内部不相交。

ST_Within在新窗口打开 - 如果几何体A完全位于几何体B内，则返回true
