### 1.常用类型



#### 1.字符串

```javascript
@param {string} str 字符串
```

#### 2.数值

```javascript
@param {number} num 数值
```

#### 3.对象

```javascript
@param {Object} obj 对象
```

#### 数组

```javascript
@param {Array} arr 数组
@param {Array.<number>} arrNum 数值数组
```

#### 任意类型

```javascript
@param {any} any 任意类型
@param {*} any 任意类型
```



#### 4.模块

```javascript
@param {module:ol/Coordinate} coord 模块
```

#### 其他类型

```
@param {HTMLElement} el html元素
```
