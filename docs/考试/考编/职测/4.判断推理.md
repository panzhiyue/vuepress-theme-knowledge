# 判断推理

## 一、图形推理

### （一）题型

- 一组图：从左往右找规律
- 两组图：以完整图为基准找规律，补充第二组图
- 九宫格：先横着看，找不到规律再竖着看
- 分组分类：给6个图，按共同规律进行分组
- 空间类：折纸盒问题

### （二）规律

#### 1.位置规律

**特征**

- 元素组成相同
  - 长相相同
  - 数量一样

**考点**

- 平移
  - 方向
    - 直线
      - 上下
      - 左右
    - 绕圈
      - 顺时针
      - 逆时针
  - 步数
    - 恒定
    - 递增（等差）
  - 技巧
    - 多个元素分开看，边做边排除
    - 位置规律-就近走原则
    - “空”可以是一种隐性移动元素
    - 剩下两个选项，可以对比选项差异
    - 先判断方向，再确定路径
      - 方向
        - 每行数量不变，按行（左右）走
        - 每列数量不变，按列（上下）走
      - 路径
        - 循环（从头开始）
        - 反弹（折返）
- 旋转
  - 方向
    - 顺时针
    - 逆时针
  - 常见角度
    - 30,45,60,90,180
  - 技巧
    - 一般来说一组图中旋转方向不变
- 翻转
  - 左右翻转：竖轴对策
  - 上下翻转：横轴对称

#### 2.样式规律

**特征**

- 元素组成相识

**考点**

- 遍历：缺啥补啥
  - 特征
    - 相同元素重复出现
  - 遍历类型
    - 元素，颜色
- 样式运算
  - 特征
    - 相同线条重复出现
  - 考点
    - 相加，相减
    - 求异（保留不同）
    - 求同（保留相同）
  - 技巧
    - 可能外部元素不参与运算，内部元素参与运算
    - 可能会与旋转等结合考查
    - 挑1-2根线入手
- 黑白运算
  - 特征
    - 背景图相同，相同区域颜色不同
  - 方法
    - 相同位置运算（颜色）
  - 技巧
    - 运算规则具体题目具体找
    - 黑+白与白+黑不一定相同
    - 注意区分
      - 黑块数量相同，优先位置平移
      - 黑块数量不同，优先黑白运算
    - 从选项差异入手，边做边验证

#### 3.属性规律

**特征**

- 元素组成不相同，不相识，优先属性

**考点**

- 对称性
  - 类型
    - 轴对称
      - 特征：等腰元素出现
    - 中心对称
      - 把一个图形绕着某一点旋转180°，如果它能够与另一个图形重合
      - 特征：平行四边形，N,S,Z变形图出现
    - 轴+中心对称
  - 考法
    - 区分轴对称，中心对称，轴+中心对称
    - 细化考查对称轴的方向，数量
  - 技巧
    - 先把对称轴都画出来
- 开闭性
  - 特征
    - 完整的图形留了小开口，可以考虑开闭性
  - 类型
    - 全封闭
    - 全开放
- 曲直性
  - 类型
    - 全曲线
    - 全直线
    - 曲+直
  - 技巧
    - 图形明显分为两个部分（上下/左右/内外）——优先分开看

#### ~~4.特殊规律~~

**考点**

- 功能元素
  - 如何识别：每幅图形均有相同小元素出现
  - 考察类型
    - 标记作用
      - 功能点
        - 点（交点）
        - 线（长短/曲直）
        - 角（锐角，直角，钝角，最大最小）
        - 面（大小，单独，相交）
      - 箭头（箭头指向，与功能点相似）
      - 技巧：看标记作用，不看数量
- 图形间关系
  - 特征：每幅图都出现两个或多个封闭图形
  - 分类
    - 相离（包括内离）
    - 相交
      - 交于点
      - 交于边
      - 交于面
    - 相较于边的细化
      - 特征：所有图形均相交于边
      - 考法
        - 看数量：有几条
        - 看样式：长/短，整体/部分,曲直

#### 5.数量规律（50%）

**特征**

- 元素组成不相同，不相识，且无属性规律，数量规律明显



**考点**

- 面
  - 什么是面
    - 空白的封闭区域（窟窿）
    - 面是白的，黑的不是
  - 面的特征图
    - 图形被分隔
    - 封闭面明显
  - 细化考法
    - 细化观察面的形状
      - 出现数面特征图，单整体数面无规律
      - 考法
        - 所有面（都是三角形，四边形等）
        - 部分面（相同形状的面，某一形状的面）
        - 单个面（最大/最小）
    - 面的数量+位置
- 线
  - 直线
    - 特征
      - 单一直线，多边形
    - 细化
      - 线的方向
        - 横竖分开数
      - 线的位置
        - 内外分开数
  - 曲线
    - 特征
      - 曲线图形（全曲线图形，圆，弧）
  - 笔画数
    - 特征
      - 出现数线特征图，但是整体数线无规律
    - 一笔画：图形由一笔画成，线条不能重复来回画
      - 线条之间全部连通
      - 奇点（发出奇数条线的点）数为0个或2个
    - 多笔画
      - 特征图
        - 经典款：五角星，日字，田字，切圆
        - 变形款
        - 潮流款：多端点，明显能一笔画成的图形
      - 笔画数
        - 奇点数/2(奇点数一定是一个偶数)
      - 多部分图形的笔画数：将每个部分的笔画数分别数出来，再相加
  - 技巧
    - 单独看没有规律，可以考虑运算（曲线数-直线数规律）
    - 图形均有外框——内外分开数
- 点
  - 线与线的交点
  - 特征图
    - 线条交叉明显（大树杈）
    - 乱糟糟-团线交叉
    - 相切较多
- 素
  - 小元素
    - 特征图：多个长得不一样，独立的小元素
    - 元素数量
    - 元素种类
  - 部分数（线条与线条连接在一起叫一部分）
    - 特征：生活化图形，粗线条图形
- 角
  - 0-180之间的角

#### 6.空间规律

##### （1）六面体

- 方法

  - 相对面
    - 在立体图中，一组相对面同时出现则排除
    - 判断
      - 同行或同列相隔一个面
      - Z字形两端（近邻中线）

  - 相邻面
    - 构成直角的两个边是同一条边
    - 应用：折叠前后相邻关系保持不变

  - 画边法
    - 结合选项，找一个特殊面的唯一点（定点）
    - 顺时针方向画边，画同一个面
    - 题干与选项对应面不一致——排除

- 所有方法都是排除错误选项的
- 折叠方向：外表面

## 二、类比推理（65%）

**题型**

- 两词型——A：B
- 三词型——A：B：C
- 填空型——A对于（     ）相当于（     ）对于B

**大纲要求**

- 给出一组相关的词，要求通过观察分析，在备选答案中找出一组与之在逻辑关系上最为贴近或相似的词

**注意**

- 逻辑关系：考察逻辑思维，不是语感
- **最**为贴近或相似：需要对比择优



### （一）语义关系

#### 1.近意关系，反义关系

##### （1）近义关系：两个词语意思相近

- 开心：高兴
- 言不由衷：口是心非

##### （2）反义关系：两个词语意思相反

- 勇敢：懦弱
- 言而有信：言而无信

**二级辨析**

- 程度加深
  - 比如“失望” “绝望”是近义关系。“失望”指失去了希望、希望落空了，但是“绝望”更加 严重，指完全断绝、失去了希望，对比发现，“绝望”比“失望”程度更深。

- 感情色彩
  - 褒义
  - 贬义
  - 中性词

#### 2.比喻义，象征义

把一种事物比喻成另外的事物，或者词语本身的含义同时是另一种事物的象征

月亮∶玉盘：“月亮”像“玉盘”，二者是比喻义。

做题时无需区分比喻义和象征义，经常会放在一起考查，只要识别出是 比喻象征义即可。

### （四）做题方法——词语拆分

**什么时候用**

- 单字重复出现：如“头∶包工头”，“包工头”中“头”指领导、首 领，是“头”本意之外的引申义。
- 成语被拆分：如“南征∶北战”，本来是完整的成语，但被拆开，“南 征”与“北战”是并列关系。再如“唇亡∶齿寒”，因为“唇亡（嘴唇没了）” 所以“齿寒（冻牙）”，二者是因果对应关系。
- 两组词语之间没有明显逻辑关系：如“成败∶呼吸”，“成败”与“呼 吸”没有关系，拆开看，“成”与“败”，“呼”与“吸”是反义词。



**因果关系总结**

- 人逢喜事∶精神爽                   城门失火∶殃及池鱼
- 众人拾柴∶火焰高                   多行不义∶必自毙
- 近水楼台∶先得月                   上梁不正∶下梁歪
- 水落石出、水涨船高、春暖花开、风吹草动、天寒地冻

## 三、逻辑判断

### （一）翻译推理

**题目特征**

- 题干和选择中存在明显的逻辑关联词

**提问方式**

- 可以推出，不能推出

**解题思路**

- 先翻译：将题干的句子翻译为A->B,B->C的形式
- 后推理：根据相关定理推理公式，对“A->B”进行推理
- 注意：不要通过理解语义来解题

**翻译规则**

- 前推后
  - 典型关联词
    - 如果.......，那么.......
    - 若.........，则...........
    - 只要........，就..........
    - 所有..........，都.............
    - ..............离不开................
  - 关联词圈起来，剩下的内容：前半句->后半句
  - 解题步骤梳理
    - 对题干进行翻译
    - 对选项进行处理，在题干中到选项出现的两个词，注意无中生有
    - 看选项箭头前的词，对题干否前/坑后推不出必然结论
- 后推前
  - 典型关联词
    - 只有.............，才..........
    - 不..........不..............
    - 除非.............，否则不..............
  - 关联词圈起来，剩下的内容，后半句->前半句
  - 关联词变形
    - 除非A否则B			-B->A
    - 基础/前提/先决条件/必不可少/必要条件/必要条件
      - 谁是必不可少，谁放在箭头后
- 推理方式
  - 重结构，轻内容
  - 题干可翻译，看后面的推理是肯前还是否前
  - 以下与题干所犯的逻辑错误相同的是

**推理规则**

- **逆否等价（重要）**
  - 符号表示：A->B=-B->-A
  - 口诀
    - 肯前比肯后，否后必否前
    - 否前肯后无必然结论/不确定（可能，可能不）
- 传递规则
  - A->B,B->C可推出A->B->C



**且和或**

- A且B
  - 两者同时成立（两真比真，一假就假）
  - 等价关键词:和,既....又...,不仅...而且...,.....但是....
- A或B
  - 两者至少一个成立（一真就真，两假才假）
  - 等价关键词：或者，或者.....或者......，至少一个
  - 推理
    - 否1->推1
    - 或关系为真的前提下,否定其中任意一项可以得到另一项一定为真
- 摩根定律
  - -（A且B）=-A或-B
  - -（A或B）=-A且-B
  - 口诀：去括号，逐项添加“-”，“且”，“或”互变



**有的**

- 概念
  - ≥1
  - 有的A是B≠有的A不是B
- 关联词：有的，有些，一些，某些，有一些
- 推理规则
  - 有的A是B=有的B是A
  - 有的A不是B=有的不是B的是A
  - “有的”二字必须放在箭头前
  - 带“有的”的翻译不适用逆否等价规则
  - 所有A都是B=有的B是A



**并非**

- “并非”一出现，后面全部否：
- “必然”换可能，“有的”换“所有”
- “且”或“互换”，谓语动词最后否



**推理结构相似**

- 特征：
  - 提问“推理结构最为相似”，“推理形式最为相似”
- 逻辑一直是前提，句式相同更优先，不要考虑推理是否正确

### （二）组合排列

**题目特征**

- 题干给出两组以上对象
- 给出几组对象之间的关系

**方法**

- 排除法

  - 何时用：题干条件为真，选项信息充分
  - 如何用：读一句，排一句
    - 快速找到“谁是谁”
    - 快速定位“谁不是谁”

- 代入法

  - 何时用
    - 题干条件有真有假
    - 提问为“可能/不可能是”
  - 如何用：把选项代入题干去验证

- 最大信息

  - 题干条件中出现次数最多的词
  - 以此作为推理起点

- 辅助技巧

  - 符号
    - ＞，＜
    - 往往涉及年龄，成绩，收入，身高等大小比较
  - 画表格
    - 排序题/几个对象，3个以上信息
    - 列表之后，优先填入确定信息

  

  

### （三）真假推理

**题目特征**

- 题目中给出若干个命题，这些命题中有真有假，要求通过判断命题的真假推理某些结论

**解题思路**

- 梳理题干
- 找矛盾关系
  - A和-A
  - 所有A都是B，有的A不是B
  - 所有A都不是B，有的A是B
  - A->B和A且-B
- 看其余

### （四）逻辑论证（高度重视）

#### 基础

**论证的三要素**

- 论据（疫情期间居家办公）----可能有
- 论证（疫情期间居家办公就是好企业）-----隐含其中
- 论点（粉笔是个好企业）-----一定有

**找论点**

- 结合提问方式-----问啥找啥
- 典型提示词:所以，结论是，这表明/说明/意味着,由此推出/可知,据此认为
- 首尾句原则

**找论据**

- 论据提示词：由于，因为，鉴于，根据
- 论据常见位置：就近原则
- 论据常见形式：原因，数据，事例，实验或调查内容等

**技巧**

- 下定义，提问题，说现象，但之前都可略读或不读
- 论点有比较关系，正确答案往往也有比较关系
- 论证题要比着来（必然>可能）
- 题目中有特殊限制时，正确选项往往会对此作出回应！



#### 1.削弱题型

##### （1）否定论点（力度最强&最常考）

**选项特征**

- 与论点表述的意思相反（表明论点是错的）

**题型特征**

- 只有论点没有论据
- 论点与论据话题一致

排除跑题项，不明确项

**解题步骤**

- 分析论点论据之间的关系
  - 只有论点，无论据
  - 论点，论据话题一致
- 寻找对应选项

##### （2）拆桥（力度仅次于否定论点）

**选项特征**

- 同时包含论点和论据中的关键词，并否定论点和论据之间的关系

**用法**

- 去同存异，说两者没关系

**什么时候考虑拆桥**

- 提问方式为“削弱论证”时
- 论点与论据话题不一致

##### （3）否定论据

**何时削弱论据**

- 观点为预测，建议类等
- 没有论点和拆桥选项

##### （4）因果倒置

将论点中的因果关系顺序颠倒（出现就优选）

##### （5）他因削弱

在原来的基础上，加入了一个新的原因也可能推出结果

##### （6）总结

**解题步骤**

- 判断题型
  - 论点论据话题一致或只有论点(否论点)
  - 论点论据话题不一致（拆桥）
  - 论点是预测，建议类（否论据）
  - 论点有因果，考虑因果倒置和它因削弱
- 找出论点论据
- 预设答案
- 对比选项

**力度比较**

- 否定论点>拆桥>否定论据>他因削弱
- 因果倒置>否定论据>他因削弱

#### 2.加强题型

##### （1）搭桥

**什么时候用**

- 提问方式为前提，假设，必要条件，加强论证时，优先考虑搭桥
- 论点和论据话题不一致

**选项特征**

- 同时包含论点和论据中的关键词，并肯定论点和论据之间的关系

**用法**

- 去同存异，说两者有关系！

##### （2）补充论据

**什么时候用**

- 必要条件：选项为论点成立所必须的（没它不行）
- 搭桥备胎

**解释**

- 说明论点成立的原因

**举例**

- 证明论点成立的例子

##### （3）总结

**文段特征**

- 论点论据话题不一致：优先想搭桥
- 只有论点/论点论据话题一致：优先想补充论据（解释，举例）

**特殊问法**

- 前提，假设，必要条件，加强论证时，优先想搭桥，其次必要条件

**力度比较**

- 搭桥=必要条件>解释>举例

### （五）原因解释

以下哪项如果为真，最能解释/不能解释上述现象（矛盾）

**解题思维**

- 找现象或矛盾（转折词：但是，然而）
- 给理由-能够解释矛盾双方
