# 判断推理

## 一、图形推理

### （一）题型

- 一组图：从左往右找规律
- 两组图：以完整图为基准找规律，补充第二组图
- 九宫格：先横着看，找不到规律再竖着看
- 分组分类：给6个图，按共同规律进行分组
- 空间类：折纸盒问题

### （二）规律

#### 1.位置规律

**特征**

- 元素组成相同
  - 长相相同
  - 数量一样

**考点**

- 平移
  - 方向
    - 直线
      - 上下
      - 左右
    - 绕圈
      - 顺时针
      - 逆时针
  - 步数
    - 恒定
    - 递增（等差）
  - 技巧
    - 多个元素分开看，边做边排除
    - 位置规律-就近走原则
    - “空”可以是一种隐性移动元素
    - 剩下两个选项，可以对比选项差异
    - 先判断方向，再确定路径
      - 方向
        - 每行数量不变，按行（左右）走
        - 每列数量不变，按列（上下）走
      - 路径
        - 循环（从头开始）
        - 反弹（折返）
- 旋转
  - 方向
    - 顺时针
    - 逆时针
  - 常见角度
    - 30,45,60,90,180
  - 技巧
    - 一般来说一组图中旋转方向不变
- 翻转
  - 左右翻转：竖轴对策
  - 上下翻转：横轴对称

#### 2.样式规律

**特征**

- 元素组成相识

**考点**

- 遍历：缺啥补啥
  - 特征
    - 相同元素重复出现
  - 遍历类型
    - 元素，颜色
- 样式运算
  - 特征
    - 相同线条重复出现
  - 考点
    - 相加，相减
    - 求异（保留不同）
    - 求同（保留相同）
  - 技巧
    - 可能外部元素不参与运算，内部元素参与运算
    - 可能会与旋转等结合考查
    - 挑1-2根线入手
- 黑白运算
  - 特征
    - 背景图相同，相同区域颜色不同
  - 方法
    - 相同位置运算（颜色）
  - 技巧
    - 运算规则具体题目具体找
    - 黑+白与白+黑不一定相同
    - 注意区分
      - 黑块数量相同，优先位置平移
      - 黑块数量不同，优先黑白运算
    - 从选项差异入手，边做边验证

#### 3.属性规律

**特征**

- 元素组成不相同，不相识，优先属性

**考点**

- 对称性
  - 类型
    - 轴对称
      - 特征：等腰元素出现
    - 中心对称
      - 把一个图形绕着某一点旋转180°，如果它能够与另一个图形重合
      - 特征：平行四边形，N,S,Z变形图出现
    - 轴+中心对称
  - 考法
    - 区分轴对称，中心对称，轴+中心对称
    - 细化考查对称轴的方向，数量
  - 技巧
    - 先把对称轴都画出来
- 开闭性
  - 特征
    - 完整的图形留了小开口，可以考虑开闭性
  - 类型
    - 全封闭
    - 全开放
- 曲直性
  - 类型
    - 全曲线
    - 全直线
    - 曲+直
  - 技巧
    - 图形明显分为两个部分（上下/左右/内外）——优先分开看

#### 4.特殊规律

**考点**

- 功能元素
  - 如何识别：每幅图形均有相同小元素出现
  - 考察类型
    - 标记作用
      - 功能点
        - 点（交点）
        - 线（长短/曲直）
        - 角（锐角，直角，钝角，最大最小）
        - 面（大小，单独，相交）
      - 箭头（箭头指向，与功能点相似）
      - 技巧：看标记作用，不看数量
- 图形间关系
  - 特征：每幅图都出现两个或多个封闭图形
  - 分类
    - 相离（包括内离）
    - 相交
      - 交于点
      - 交于边
      - 交于面
    - 相较于边的细化
      - 特征：所有图形均相交于边
      - 考法
        - 看数量：有几条
        - 看样式：长/短，整体/部分,曲直

#### 5.数量规律（50%）

**特征**

- 元素组成不相同，不相识，且无属性规律，数量规律明显



**考点**

- 面
  - 什么是面
    - 空白的封闭区域（窟窿）
    - 面是白的，黑的不是
  - 面的特征图
    - 图形被分隔
    - 封闭面明显
  - 细化考法
    - 细化观察面的形状
      - 出现数面特征图，单整体数面无规律
      - 考法
        - 所有面（都是三角形，四边形等）
        - 部分面（相同形状的面，某一形状的面）
        - 单个面（最大/最小）
    - 面的数量+位置
- 线
  - 直线
    - 特征
      - 单一直线，多边形
    - 细化
      - 线的方向
        - 横竖分开数
      - 线的位置
        - 内外分开数
  - 曲线
    - 特征
      - 曲线图形（全曲线图形，圆，弧）
  - 笔画数
    - 特征
      - 出现数线特征图，但是整体数线无规律
    - 一笔画：图形由一笔画成，线条不能重复来回画
      - 线条之间全部连通
      - 奇点（发出奇数条线的点）数为0个或2个
    - 多笔画
      - 特征图
        - 经典款：五角星，日字，田字，切圆
        - 变形款
        - 潮流款：多端点，明显能一笔画成的图形
      - 笔画数
        - 奇点数/2(奇点数一定是一个偶数)
      - 多部分图形的笔画数：将每个部分的笔画数分别数出来，再相加
  - 技巧
    - 单独看没有规律，可以考虑运算（曲线数-直线数规律）
    - 图形均有外框——内外分开数
- 点
  - 线与线的交点
  - 特征图
    - 线条交叉明显（大树杈）
    - 乱糟糟-团线交叉
    - 相切较多
- 素
  - 小元素
    - 特征图：多个长得不一样，独立的小元素
    - 元素数量
    - 元素种类
  - 部分数（线条与线条连接在一起叫一部分）
    - 特征：生活化图形，粗线条图形
- 角
  - 0-180之间的角

#### 6.空间规律

##### （1）六面体

- 方法

  - 相对面
    - 在立体图中，一组相对面同时出现则排除
    - 判断
      - 同行或同列相隔一个面
      - Z字形两端（近邻中线）

  - 相邻面
    - 构成直角的两个边是同一条边
    - 应用：折叠前后相邻关系保持不变

  - 画边法
    - 结合选项，找一个特殊面的唯一点（定点）
    - 顺时针方向画边，画同一个面
    - 题干与选项对应面不一致——排除

- 所有方法都是排除错误选项的
- 折叠方向：外表面
