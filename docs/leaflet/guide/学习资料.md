# Leaflet学习资料

源码地址:https://github.com/Leaflet/Leaflet

官方文档:https://leafletjs.com/

CDN:写示例时更方便

```html
    <!-- leaflet样式 -->
    <link href="https://unpkg.com/leaflet@1.7.1/dist/leaflet.css" rel="stylesheet" />
    <!-- leaflet类库 -->
    <script src="https://unpkg.com/leaflet@1.7.1/dist/leaflet-src.js"></script>
```
