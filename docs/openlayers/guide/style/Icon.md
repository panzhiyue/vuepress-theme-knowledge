## Anchor控制图标位置
```
[x,y]   x表示横轴(0:最左边,1:最右边)，y表示纵轴(0:最上面,1:最下面)
[0,0]   原点位置在图片左上角
[1,0]   原点位置在图片右上角
[0, 1]  原点位置在图片左下角
[1, 1]  原点位置在图片右下角

```
