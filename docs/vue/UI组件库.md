# UI组件库

## element-ui

1.安装

```
npm i -S element-ui
```

2.在代码中引用

```
import Vue from "vue";
import ElementUI from 'element-ui';
import 'element-ui/lib/theme-chalk/index.css';
import App from "./App.vue";
import router from "./router";

//使用ElementUI
Vue.use(ElementUI);
```
